Welcome to the Clarango resource page.

On this page we would like to give you a comprehensive overview of the Clarango documentation to get you started as fast as possible.

  • First browse the located on GitHub: Clarango (see button top right)
  • You can also clone the project fast using: git clone https://github.com/edlich/clarango.git
  • Next you can start coding by inserting [clarango "0.3.1"] into your project.clj file and then require the namespaces you need: 
    (:require [clarango.core :as clcore]
         [clarango.database :as database])
  • Then have a look at the complete API doc.
  • For a hands on extensive guide see the Clarango book (coming soon).
